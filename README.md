SQL Project Portfolio:

1. Items & Product Sales Database
   
Project Description

This project simulates a product sales database where item details and product sales data are stored in separate databases. SQL queries were used to manage product information, analyze sales performance, and perform relational data operations between item and product tables.

The database includes information such as product sales, manufacturing price, gross sales, profit, vendor details, and item pricing.

Key Tasks:

Created Brands and Products databases
Built tables for product and item details
Inserted product sales records
Performed filtering and aggregation queries
Implemented JOIN operations between tables
Created stored procedures and triggers

Example Insights:

Identified highest profit generating products
Calculated total sales and gross sales
Analyzed country-wise product performance
Retrieved top 3 most expensive items

6. Employee Salary & Manager Analysis
   
Project Description:

This project analyzes employee and salary datasets to demonstrate advanced SQL query techniques including subqueries, window functions, and set operations.

Key SQL Operations:

Finding employees without project assignments
Calculating project-wise employee counts
Retrieving highest or nth highest salary values

Example Insights:

Retrieved third highest salary without using TOP
